# Simon Says Game

[Live Demo](https://ruturajgawade.github.io/Simon-Says-Game/)

A web-based implementation of the classic memory game **Simon Says**, built using **HTML**, **CSS**, and **JavaScript**.

---

## About

Simon Says is a memory game where the player must repeat increasingly long sequences of tones and colors. Each round, a new step is added to the sequence, and the player must replicate it by clicking the colored buttons in the correct order.

This project is a web-based version of that game, designed to run in a browser with a clean and responsive interface.

## How to Play

1. Open the game in your browser.  
2. Click **Start** to begin.  
3. Watch the sequence of colors lighting up.  
4. Repeat the sequence by clicking on the same colored buttons in the same order.  
5. If you succeed, the sequence gets longer. If you make a mistake, the game ends.  
6. Try to remember and repeat as many steps as possible to get a high score.
